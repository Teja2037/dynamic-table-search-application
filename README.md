# Dynamic Table Search Application

A simple and interactive web application developed using HTML and JavaScript that displays cricket team data in a table and allows users to search and filter records dynamically. The project demonstrates JavaScript array manipulation, DOM updates, event handling, and real-time search functionality.

## Features

* Display cricket team information in a structured table
* Real-time search and filtering
* Dynamic table updates without page refresh
* User-friendly search interface
* Lightweight and fast-loading application
* Beginner-friendly JavaScript project

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Run

1. Download or clone the repository.
2. Open `filter.html` in any modern web browser.
3. Enter a keyword in the search box.
4. View the filtered results instantly in the table.

## Purpose

This project was developed to practice front-end web development concepts such as data filtering, array methods, event handling, DOM manipulation, and real-time search functionality using HTML and JavaScript.

## Learning Outcomes
